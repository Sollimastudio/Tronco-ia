# Memoria de Projeto

A memoria de projeto e o documento vivo que impede a IA de se perder.

Campos principais:

- nome do projeto
- tipo de produto
- publico alvo
- promessa central
- voz e tom
- estetica visual
- arquivos anexados
- decisoes aprovadas
- etapa atual
- blocos produzidos
- arquivos gerados
- proximos passos

Regra: ao final de cada etapa grande, o sistema atualiza a memoria antes de seguir.
